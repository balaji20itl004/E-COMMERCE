DC3.3V-5V UV Detection Sensor Module For Arduino Sensor

DESCRIPTION

The DC 3.3V-5V UV Detection Sensor Module for Arduino is a compact, efficient device designed to detect ultraviolet (UV) light intensity in the environment. It operates on a supply voltage range of 3.3V to 5V, making it compatible with most microcontroller platforms, including Arduino. The module features high sensitivity to UV light and outputs an analog signal proportional to the UV intensity, allowing easy integration into projects like UV monitoring, weather stations, and wearable devices. Its small size and straightforward interface make it ideal for both beginners and advanced users working on electronics and IoT applications.

  • SPECIFICATIONS
    • Operating Voltage: 3.3V to 5V
    • Detection Range: Ultraviolet light (UV)
    • Output Type: Analog signal proportional to UV intensity
    • Compatibility: Arduino and other microcontrollers
    • Applications: UV monitoring, flame detection, sunlight analysis
    • Compact and easy to integrate
    • High sensitivity photodiode for UV detection
